Study Plan for Web Development Module

Module Overview

The Web Development module is designed to provide students with a comprehensive understanding of the core concepts and principles of web development.

Module Content

HTML

  • Introduction to HTML
  • HTML Elements and Attributes
  • HTML Headings and Paragraphs
  • HTML Links and Images
  • HTML Forms and Tables

CSS

  • Introduction to CSS
  • CSS Syntax and Selectors
  • CSS Text and Font Properties
  • CSS Box Model and Layout
  • CSS Colors and Backgrounds

JavaScript

  • Introduction to JavaScript
  • JavaScript Variables and Data Types
  • JavaScript Functions and Loops
  • JavaScript Events and DOM Manipulation
  • JavaScript Forms and Validation

Module Assessment

The assessment for this module will consist of a written exam and a practical project.
