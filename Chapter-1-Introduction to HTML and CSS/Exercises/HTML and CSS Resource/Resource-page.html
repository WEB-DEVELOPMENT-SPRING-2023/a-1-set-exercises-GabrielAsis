HTML & CSS Beginner Guide

HTML

HTML stands for Hypertext Markup Language. It is a markup language used to create and design web pages. HTML allows web developers to structure content on a web page by using a set of predefined tags and attributes. These tags and attributes provide instructions to web browsers on how to display content, such as text, images, videos, and links, on a web page.

Basic HTML Tags:

HTML Tag

<html>
</html>

This tag indicates the start and end of an HTML document.

Head Tag

<head>
</head>

This tag contains metadata about the web page, such as the title, links to CSS stylesheets, and scripts. Anything written inside this tag will not be displayed on the webpage

Title Tag

<title></title>

This tag defines the title of the web page and appears ONLY in the browser's title bar.

Body Tag

<body>
</body>

This tag defines the main content of the web page. This is where you will be able to display elements on to your webpage

This is how a simple HTML document should look like when using the tags above:

<html>
  <head>
    <title>Title of Page</title>
  </head>
  <body>
    Content of Page...
  </body>
</html>

Heading 1 to Heading 6 Tags

<h1></h1>
<h2></h2>
<h3></h3>
...

These tags define different levels of headings, with <h1> being the largest and most important heading.

Example:

Code:

<h1>Heading 1</h1>

<h2>Heading 2</h2>

<h3>Heading 3</h3>

<h4>Heading 4</h4>

<h5>Heading 5</h5>

<h6>Heading 6</h6>

Output:

Heading 1

Headin 2

Heading 3

Heading 4

Heading 5
Heading 6

Paragraph / P tag

<p></p>

This tag defines a paragraph of text.

Example:

Code:

<p> My name is Walter Hartwell White</p>

Output:

My name is Walter Hartwell White

A tag

<a></a>

This tag creates a hyperlink to another web page or a specific section of a web page.

Example:

Code:

<a href="https://google.com/">Google Link</a>

Output:

Google Link

Image / img tag

<img>

This tag inserts an image into the web page. You can use a file from your computer or copy an image link from the internet.

Example:

Code:

<img src="https://cdn.shopify.com/s/files/1/0583/1006/6338/files/og-id.jpg" width="400">

Output:

Ordered List / ol tag

<ol>
  <li></li>
</ol>

These tags are used to create an ordered list of items meaning the items are listed in a specific order.

Example:

Code:

<ol>
  <li>Kendrick Lamar</li>
  <li>Tyler, The creator</li>
  <li>Kanye West</li>
</ol>

Output:

  1. Kendrick Lamar
  2. Tyler, The creator
  3. Kanye West

Unordered List / ul tag

<ul>
  <li></li>
</ul>

These tags are used to create an unordered list of items meaning the items are listed in no particular order.

Example:

Code:

<ul>
  <li>Apple</li>
  <li>Soap</li>
  <li>Chicken</li>
  <li>Rice</li>
</ul>

Output:

  • Apple
  • Soap
  • Chicken
  • Rice

CSS

CSS is a language used for describing the style and formatting of web pages written in markup languages like HTML. It allows web developers to define the look and layout of web page content, such as font styles, colors, and positioning, separately from the content itself. This separation of content and presentation helps developers create consistent and visually appealing web pages across a website.

Basic CSS Properties:

Color

Sets the text color of an element.

Example:

Code:

.red{
color: red;
}

.blue{
color: blue;
}

.green{
color: green;
}

Output:

This text is red

This text is red

This text is red

Background-color

Sets the background color of an element.

Example:

Code:

.bg1{
background-color: pink;
}

.bg2{
background-color: lightblue;
}

.bg3{
background-color: lightgreen;
}

Output:

The background is pink

The background is light-blue

The background is light-green

Font-size

Sets the size of the text inside an element.

Example:

Code:

.p1{
font-size: 15px        
        }

.p2{
font-size: 50px        
}

.p3{
font-size: 25px        
}

Output:

Font size = 15px

Font size = 50px

Font size = 30px

Font-family

Sets the font family used for the text inside an element.

Example:

Code:

.p-arial{
font-family: Arial, Helvetica, sans-serif;
}
                
.p-impact{
font-family: Impact, Haettenschweiler, 'Arial Narrow Bold', sans-serif;        
}
                
.p-cambria{
font-family: Cambria, Cochin, Georgia, Times, 'Times New Roman', serif       
}

Output:

Font-family: Arial

Font-family: Impact

Font-family: Cambria

Padding

Sets the space between an element's content and its border.

Example:

Code:

<img src="https://cdn.shopify.com/s/files/1/0583/1006/6338/files/og-id.jpg" width="400px">

Output:

Box 1

Box 2

Box 3

Margin

Sets the space between an element's border and surrounding elements.

Example:

Code:

.marg1{
margin-bottom: 50px;
}

.marg2{
margin-top: 30px;
margin-left: 60px;
}

.marg3{
margin-top:100px;
margin-left:500px; 
}

Output:

First Sentence.

Second Sentence.

Third Sentence.

Border

Sets the border around an element.

Example:

Code:

.border1{
border: 3px solid red;
padding: 10px;
}

.border2{
border: 10px solid blue;
padding: 25px;  
}

.border3{
border: 1px solid green;
padding: 5px;
}

Output:

Border 1

Border 2

Border 3

Text-align

Sets the alignment of the text inside an element.

Example:

Code:

.center{
text-align: center;
}

.right{
text-align: right;
}

Output:

Center

Right

Default
